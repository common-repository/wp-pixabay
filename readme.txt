== Description ==

**WPC Pixabay** helps you search millions of free photos, vectors and art illustrations from pixabay.com then insert into content or set as featured image very quickly.

= Features =

- Work with Gutenberg editor
- Search images with powerful filters
- Quick insert image with original URL
- Use image as featured image
- Multiple select support (classic editor)
- WPML integration
- Upload image(s) to Media Library (Premium Version)

== Installation ==

1. Unzip the wp-pixabay.zip
2. Copy wp-pixabay folder to wp-content/plugins
3. Go to Installed Plugins, find WPC Pixabay and click Active
4. Now when you create/edit the post, you can see the Pixabay button above the editor
5. Enjoy!
